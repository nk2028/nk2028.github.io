Ngiox Khyen 2028 項目

項目簡介

我們開發有關語言學的項目，尤其是有關音韻學，異體字和日語語言學的項目。

加入我們

若閣下對我們的工作感興趣，請傳送電郵至 GitHub 主頁上顯示的電郵地址，並附註閣下的 GitHub 帳戶。管理員很快就會將閣下添加為成員。

項目

音韻學

  • qieyun-encoder (repo): 將切韻音系音韻地位轉換為三字符音韻編碼與音韻描述
  • qieyun-text-label (repo): 標註切韻音系音韻地位的中文文本語料庫
  • qieyun-data (repo): 漢字的切韻音系音韻地位資料庫，原始資料來自廣韻
  • qieyun-sqlite (repo): 切韻音系 SQLite 資料庫
  • qieyun-js (repo): 切韻音系 JavaScript 函式庫
  • purescript-qieyun (repo): 切韻音系 PureScript 函式庫
  • qieyun-examples (repo): 使用 qieyun-js 生成切韻音系衍生品的 JavaScript 程式碼實例
  • qieyun-examples-node (repo): qieyun-examples 的 Node.js 版本
  • qieyun-autoderiver (repo | app): 切韻音系自動推導器
  • qieyun-tools (repo | app): 音韻地位查詢器，使用音韻地位表達式查找對應漢字
  • qieyun-extrapolation-tools (repo): 根據切韻音系推導後世漢字音演化規律時使用的工具（含廣州音、日語漢音等）
  • pyanxchet (repo | app): 自動反切器，由反切自動得到切韻音系音韻地位的工具
  • ToMiddleChinese (repo): 為中文文本標註中古漢語擬音的 Python 函式庫

異體字

  • yitizi (repo): 輸入一個漢字，輸出它的全部異體字
  • opencc-js (repo): 純 JavaScript 版本實作的 OpenCC
  • opencc-web (repo | app): 線上 OpenCC 轉換器

其他

  • rime-utils (repo): 用於操作 rime 配置及詞典的實用工具
  • putonghua-ipa-converter (repo | app): 为中文文本标注普通话发音的国际音标的工具
  • onyomi-gakken-kanwa (repo): 《學研漢和大字典》音讀資料
