The Ngiox Khyen 2028 Project

Project Overview

We develop projects about linguistics, especially about historical Chinese phonology, variant Chinese characters and Japanese linguistics.

Projects

Historical Chinese Phonology

  • qieyun-encoder: A encoder for converting a phonological position of the Qieyun phonological system to the three-character phonological encoding and the phonological description
  • qieyun-text-label: A corpus of Chinese texts labelled with their phonological position of the Qieyun phonological system
  • qieyun-data: A database of the phonological position of the Qieyun phonological system of Chinese characters, based on Guangyun
  • qieyun-sqlite: A/an SQLite database for the Qieyun phonological system
  • qieyun-js: A JavaScript library for the Qieyun phonological system
  • qieyun-examples: JavaScript code examples to generate the derivatives of the Qieyun phonological system using qieyun-js
  • qieyun-examples-node: Node.js version of qieyun-examples
  • qieyun-autoderiver: An automatic tool to generate the derivatives of the Qieyun phonological system
  • qieyun-tools: A tool to find the Chinese characters by phonological expressions
  • qieyun-extrapolation-tools: A tool to find the laws of sound change of post-Qieyun phonological systems (Including Cantonese, Japanese Kan-on and others)
  • pyanxchet: A tool to automate the process of obtaining the phonological position of the Qieyun phonological system of a Chinese character by its fanqie
  • ToMiddleChinese: A Python library to add reconstructed pronunciations of Middle Chinese on Chinese texts

Variant Chinese Characters

  • yitizi: Input a Chinese character. Output all the variant characters of it
  • opencc-js: Pure JavaScript implementation of OpenCC
  • opencc-web: An online OpenCC converter

Others

  • rime-utils: Utilities for operating rime configurations and dictionaries
  • onyomi-gakken-kanwa: Onyomi data on the Gakken Kanwa Large Dictionary

項目

漢語音韻學

  • qieyun-encoder: 將切韻音系音韻地位轉換為三字符音韻編碼
  • qieyun-text-label: 標註切韻音系音韻地位的中文文本語料庫
  • qieyun-data: 漢字的切韻音系音韻地位資料庫，原始資料來自廣韻
  • qieyun-sqlite: 切韻音系 SQLite 資料庫
  • qieyun-js: 切韻音系 JavaScript 函式庫
  • qieyun-examples: 使用 qieyun-js 生成切韻音系衍生品的 JavaScript 程式碼實例
  • qieyun-examples-node: qieyun-examples 的 Node.js 版本
  • qieyun-autoderiver: 切韻音系自動推導器
  • qieyun-tools: 音韻地位查詢器，使用音韻地位表達式查找對應漢字
  • qieyun-extrapolation-tools: 根據切韻音系推導後世漢字音演化規律時使用的工具（含廣州音、日語漢音等）
  • pyanxchet: 自動反切器，由反切自動得到切韻音系音韻地位的工具
  • ToMiddleChinese: 為中文文本標註中古漢語發音的 Python 函式庫

異體字

  • yitizi: 輸入一個漢字，輸出它的全部異體字
  • opencc-js: 純 JavaScript 版本實作的 OpenCC
  • opencc-web: 線上 OpenCC 轉換器

其他

  • rime-utils: 用於操作 rime 配置及詞典的實用工具
  • onyomi-gakken-kanwa: 《學研漢和大字典》音讀資料
