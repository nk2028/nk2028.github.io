The Ngiox Khyen 2028 Project

Project Overview

We develop projects about linguistics, especially about historical Chinese phonology, variant Chinese characters and Japanese linguistics.

Projects

Historical Chinese Phonology

  • qieyun-encoder (repo): A encoder for converting a phonological position of the Qieyun phonological system to the three-character phonological encoding and the phonological description
  • qieyun-text-label (repo): A corpus of Chinese texts labelled with their phonological position of the Qieyun phonological system
  • qieyun-data (repo): A database of the phonological position of the Qieyun phonological system of Chinese characters, based on Guangyun
  • qieyun-sqlite (repo): A/an SQLite database for the Qieyun phonological system
  • qieyun-js (repo): A JavaScript library for the Qieyun phonological system
  • purescript-qieyun (repo): A PureScript library for the Qieyun phonological system
  • qieyun-examples (repo): JavaScript code examples to generate the derivatives of the Qieyun phonological system using qieyun-js
  • qieyun-examples-node (repo): Node.js version of qieyun-examples
  • qieyun-autoderiver (repo | app): An automatic tool to generate the derivatives of the Qieyun phonological system
  • qieyun-tools (repo | app): A tool to find the Chinese characters by phonological expressions
  • qieyun-extrapolation-tools (repo): A tool to find the laws of sound change of post-Qieyun phonological systems (Including Cantonese, Japanese Kan-on and others)
  • pyanxchet (repo | app): A tool to automate the process of obtaining the phonological position of the Qieyun phonological system of a Chinese character by its fanqie
  • ToMiddleChinese (repo): A Python library to add reconstructed pronunciations of Middle Chinese on Chinese texts

Variant Chinese Characters

  • yitizi (repo): Input a Chinese character. Output all the variant characters of it
  • opencc-js (repo): Pure JavaScript implementation of OpenCC
  • opencc-web (repo | app): An online OpenCC converter

Others

  • rime-utils (repo): Utilities for operating rime configurations and dictionaries
  • putonghua-ipa-converter (repo | app): A tool to add Putonghua pronunciations in IPA form on Chinese texts
  • onyomi-gakken-kanwa (repo): Onyomi data on the Gakken Kanwa Large Dictionary
