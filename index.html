Projects

Qieyun Projects

  • qieyun-sqlite: SQLite database for Qieyun
  • qieyun-js: JavaScript library for Qieyun
  • qieyun-autoderiver: Autoderiver for Qieyun
  • qieyun-examples: Examples of qieyun-js
  • qieyun-examples-node: Examples of qieyun-js (Node.js version)
  • qieyun-tools: Web utilities for Qieyun
  • qieyun-text-label: Labelling Chinese texts in various phonological systems, using qieyun-js

Traditional Chinese Phonology

  • pyanx: A tool that automates the fanqie process
  • ToMiddleChinese: A tool that converts Chinese texts to its pronunciation in Middle Chinese

Variant Chinese Characters

  • yitizi: Input a Chinese character. Output all the variant characters of it
  • opencc-js: Pure JavaScript implementation of OpenCC
  • opencc-web: A web interface for OpenCC
